# Keyboard Shortcuts

LaRecipe comes with useful keyboard shortcuts, they might be useful in some cases!

---

- [Keyboard Shortcuts](#keyboard-shortcuts)

<a name="keyboard-shortcuts"></a>
## Keyboard Shortcuts:

| Key | Action   |
| : |   :-   |
| / | Toggle SideBar |
| s | Toggle Search Box   |
| t | Scroll to Top  |
| b | Scroll to Bottom  |
